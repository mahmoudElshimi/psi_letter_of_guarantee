Installation Instructions

How to install and configure Psi Letter of Guarantee

To install the Psi Letter of Guarantee module, you can choose one of the following methods:

Clone the Repository:

Clone the Psi Letter of Guarantee module into your custom addons path: 

git clone ssh://git@github.com/mahmoudElshimi/psi_letter_guarantee.git /path/to/your/custom/addons/psi_letter_guarantee

Odoo Apps Store:

Alternatively, you can install the module directly from the Odoo Apps Store:

Psi Letter of Guarantee on Odoo Apps
